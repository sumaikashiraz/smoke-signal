# smoke-signal
# 🚬 VapeScope — U.S. E-Cigarette Retail Market Analysis (2020–2025)

A Kaggle-level data science portfolio project performing end-to-end exploratory,
econometric, and predictive analysis on 116,000+ U.S. e-cigarette retail
observations across 51 states, 10 brands, and 7 flavor categories.

## What's Inside
- Full EDA with 15+ publication-quality visualizations
- Difference-in-Differences causal analysis of flavor ban policy impact
- K-Means state clustering by regulatory-market profile
- Market shock detection using rolling Z-score anomaly detection
- Brand survival & flavor migration analysis
- XGBoost / LightGBM / Random Forest predictive models (R² ~0.97)
- SHAP explainability (global + local feature attribution)
- SARIMA + Holt-Winters 12-month sales forecasting
- Actionable insights for regulators, brands, and investors

## Stack
Python · pandas · scikit-learn · XGBoost · LightGBM · SHAP · statsmodels · Plotly · Seaborn

## Data Source
TobaccoMonitoring.org (CDC Foundation / Truth Initiative) — modeled from
published national/state retail aggregates, FDA PMTA orders, and state legislative records.
